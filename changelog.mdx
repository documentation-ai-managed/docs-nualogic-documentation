## Integrate New Features

Follow these steps to adopt the latest AI integration capabilities from v1.2.0.

<Steps>
  <Step title="Set Up API Key" icon="key">
    Generate your `{NUALOGIC_API_KEY}` from the dashboard.
  </Step>
  <Step title="Install SDK" icon="download">
    Use the following multi-language examples to get started.
  </Step>
  <Step title="Run First Automation" icon="play">
    Test with a sample AI prompt chain.
  </Step>
</Steps>

<CodeGroup tabs="Python,JavaScript">
  ```python
  import requests
  from langchain.llms import NuaLogicLLM

  api_key = "{NUALOGIC_API_KEY}"
  llm = NuaLogicLLM(api_key=api_key)
  response = llm("Optimize my SME supply chain")
  print(response)
  ```
  ```javascript
  const { NuaLogicLLM } = require('nualogic-sdk');

  const llm = new NuaLogicLLM({ apiKey: '{NUALOGIC_API_KEY}' });
  const response = await llm.generate("Optimize my SME supply chain");
  console.log(response);
  ```
</CodeGroup>
